# @squirrel-forge/ui-modal
An accessible modal with events and plugin support, made for the browser and babel compatible.

## Installation

```
npm i @squirrel-forge/ui-modal
```

## Usage

```
import { Class, function } from '@squirrel-forge/ui-modal';
```
